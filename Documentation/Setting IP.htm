Setting IP


IPBuddy allows you to either set a static IP address on a network adapter or force a network adapter back to DHCP. IPBuddy utilizes this functionality to help connect to local network controllers on potentially complex networks. Therefore, the settings for the static IP address are located in the NAE information.

Set a static IP address

To set a static IP address:

  • Select an NAE from the main list
  • In the NAE settings table, verify the values for 'Static IP', 'Static Subnet', and 'Static Gateway' are correct.
  • From the 'Network Adapters' area select the network adapter in the drop-down menu you would like to apply this IP to.
  • Click the 'Set Static' button.

To set DHCP

To force a network adapter back to DHCP:

  • From the 'Network Adapters' area select the network adapter in the drop-down menu you would like to force to DHCP
  • Click the 'Set DHCP' button
