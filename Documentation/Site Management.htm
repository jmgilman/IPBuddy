Site Management


Adding a site

There are two methods to add a new site to IPBuddy:

  • From the 'File' menu, select 'New' and then 'Site'.
  • Right-click on a blank area of the main list and select 'New Site' from the resulting menu.
      • Both of these methods will create a new site on the main list with a default name of 'Site Name' and will initially prompt you to change the name. To change the name at a later date, simply double-click the name in the list and change it accordingly.

      Removing a site

      To remove a site, right-click it in the main list and select 'Delete' from the resulting menu.

      Exporting a site

      To export a site:

      • Right-click the site you wish to export in the main list.
      • Select 'Export' from the resulting menu.
      • Enter in a file name (or accept the default name) to export to and select 'Save.'

      Importing a site

      To import a site:

      • Right-click on a blank area of the main list.
      • Select 'Import Site' from the resulting menu
      • Select the appropriate site XML file and select 'Open.'
