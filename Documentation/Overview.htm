Overview

IPBuddy at its core is a very simple program to operate. IPBuddy stores all information in a tree-like list located at the bottom of the main program panel. This list consists of sites and IP addresses which are used for organization. A site is a physical location in the world, like a customer's building. Since each site can contain multiple controllers which in turn can be on different subnets, each site in IPBuddy can have multiple IP addresses. The goal of this type of structure is to allow a technician to keep all of their commonly used IP addresses organized to the various sites they visit on a normal basis. These IP addresses are then used to directly configure a network adapter's static IP address. These concepts are summarized below:

Adding a Site

There are two methods to adding a site:

  • Click the 'File' button at the top of the program and from the resulting drop-down list select 'New' and then 'New Site'.
  • Right click on the main list and select 'New Site' from the resulting drop-down list.
Both of these options add a new site to the list with a default name of 'Site Name'. By default, the program will prompt you to edit this value to something more meaningful.

To change an existing site's name, simply double-click the site name in the list.

To delete a site, right-click the site name in the main list and select 'Delete' from the resulting drop-down menu.

To duplicate a site right-click the site name and select 'Copy' and then right-click anywhere in the main list and select 'Paste'. This will copy the site and all of its contents into a new site.

Adding a IP Address

There are two methods to adding an IP address:

  • Select the site you wish to add an IP address to in the main list. Click the 'File' button at the top of the program and from the resulting drop-down list select 'New' and then 'New IP Address'.
  • Right click on the site you wish to add an IP address to in the main list and select 'Add New IP'.
Both of these options add a new IP address to the selected site with a default name of 'New IP Address'. By default, the program will prompt you to edit this value to something more meaningful. Furthermore, the program will automatically assign generic IP settings to the new IP address which you can edit later.

To edit an IP address, select the IP address in the main list. This will automatically populate the text fields under 'Set IP Settings' with the IP address' saved values. Changing these values will automatically save them to the IP address.

To delete an IP address, right-click the IP address name in the main list and select 'Delete' from the resulting drop-down menu.

To duplicate an IP address right-click the IP address name and select 'Copy' and then right-click on the site you wish to duplicate it to and select 'Paste'. This will copy over all existing IP settings.

Setting a Static IP Address

To set a static IP address, do the following:

  • Select an IP address from the main list and ensure the fields in 'Set IP Settings' are populated with the correct addresses.
  • Under 'Network Settings' ensure the 'Select Network' box is selected to the network adapter that you wish to assign the IP address to.
  • Click the 'Set IP' button. Please note that changing IP addresses requires administrative rights on Windows machines and Windows may prompt you to allow the program to continue. Please select 'Yes' to this prompt to allow the program to work correctly.

Reverting an Adapter to DHCP

To return a network adapter back to DHCP, do the following:

  • Under 'Network Settings' ensure the 'Select Network' box is selected to the network adapter that you wish to return to DHCP.
  • Click the 'Reset to DHCP' button. Please note that changing DHCP settings requires administrative rights on Windows machines and Windows may prompt you to allow the program to continue. Please select 'Yes' to this prompt to allow the program to work correctly.

Saving Your Configured Settings

Please note that by default the program does not save your configured settings. IPBuddy stores all program settings in an XML format.

To save your settings, click the 'File' button at the top of the program and select the 'Save' option from the resulting drop-down list. This will save your current settings to the file 'default.xml' in the current program directory. When the program starts it automatically searches for this 'default.xml' file and loads it upon starting up.

To save your settings to a different file other than 'default.xml', click the 'File' button at the top of the program and select 'Save As' from the resulting drop-down list. The program will open a file dialog and prompt you for the location to save the configuration file to. Select a location and name for the file and then click 'Save'.

To load previous settings you saved in a configuration file, click the 'File' button at the top of the program and select 'Open' from the resulting drop-down list. The program will open a file dialog and prompt for the location of the previously saved configuration file. Select the file and then select 'Open' to load the configuration file into the program. Please note this overrides any existing configuration settings in the program.
